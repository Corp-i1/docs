## Prerequisites

- .NET 8 is the only tested version however with some light testing .NET 9 seems stable
- Works with any .NET project type (console, desktop, web, etc.)

## Install via NuGet Package Manager Console
```bash
NuGet\Install-Package C1Logger
```

Or for a specific version:

```bash
NuGet\Install-Package C1Logger --Version x.x.x
```


## Install via .NET CLI
```
dotnet add package C1Logger
```

## Install via Visual Studio UI

1. Right-click your project in **Solution Explorer** and select **Manage NuGet Packages**.
2. Search for `C1Logger`.
3. Click **Install**.


---

**Tip:**  
If you encounter issues, ensure your project targets a supported .NET version and that NuGet package sources are configured correctly.
